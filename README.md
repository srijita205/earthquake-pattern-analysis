# Earthquake Pattern Analysis

## Objective
Analyze global earthquake patterns using USGS data — 
frequency, magnitude distribution, depth trends, and geographic clustering.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn, Folium)
- Data Source: USGS Earthquake Catalog API

## How to Run
1. Clone the repo
2. pip install -r requirements.txt
3. python main.py
